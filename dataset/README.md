Dataset

The original dataset is not included in this repository due to redistribution restrictions.

This project used the 2024 Youth Life Survey microdata provided through MDIS.

To reproduce the analysis, please download the dataset directly from the official MDIS website.

- Dataset: 2024 Youth Life Survey
- Source: "MDIS Microdata Integrated Service" (https://mdis.mods.go.kr/ofrData/selectOfrDataDetail.do?survId=1005421&itmDiv=2&nPage=3&itemId=2002&itemNm=%EC%82%AC%ED%9A%8C%EC%9D%BC%EB%B0%98)

The dataset was used only for academic purposes as part of a Data Science Mathematics team project at Gachon University.
